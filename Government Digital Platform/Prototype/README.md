# אב-טיפוס Fidelity גבוה — פורטל שירותים ממשלתי אחיד

אב-טיפוס נווט, אינטראקטיבי, לשני המוצרים (Citizen Portal + Government Operations Platform),
מבוסס טוקני **IGDS**, **RTL**, **מודל סטטוס דו-רבדי** (D-009), 7 התבניות ו-20 ה-Templates,
ומבוסס **מחקר שטח gov.il** (ראו `Canon/08_GovIl_Field_Research.md`).

## הפעלה
- **הכי פשוט:** לחיצה כפולה על `index.html` (נפתח בדפדפן ישירות, ללא שרת).
- **או שרת מקומי:** `python3 -m http.server 8731` ואז `http://localhost:8731`.

## ניווט (גרסה 0.5 — ללא סרגל פיתוח)
- **מסך פתיחה (Landing)** הוא נקודת הכניסה — מפנה לשני סוגי המשתמשים: **אזור אישי לאזרח** ו**מערכת ניהול הבקשות**.
- בתוך כל מוצר — ניווט אמיתי בלבד: Sidebar (אזרח), Header עם מחליף מערכת (Operations), Breadcrumb, Stepper, פילטרים.
- **לחיצה על הלוגו** בכותרת מחזירה למסך הפתיחה (החלפת מוצר).
